# yuki-wechat-js-sdk
⛩  适用于webpack2的微信jsSDK

## 安装
```bash
$ yarn add yuki-wechat-js-sdk
# or
$ npm i --save yuki-wechat-js-sdk
```

## 使用
```js
import 'yuki-wechat-js-sdk'

wx.config({ ... })
```
